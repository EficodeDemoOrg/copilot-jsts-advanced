## Overview

These exercises teach you to build agentic workflows with GitHub Copilot using a working weather app as the substrate. You will **not** modify the application code — instead, you'll create Copilot customization files that enhance how Copilot understands and interacts with this codebase.

## Prerequisites

- VS Code with GitHub Copilot extension
- Node.js 20+ installed
- The weather app running locally (`npm run dev`)
- All tests passing (`npm test`)

## Exercise 1: Custom Instructions

Learn to write `.instructions.md` files with `applyTo` patterns that give Copilot context-aware guidance for different parts of the codebase.

## Exercise 2: Custom Agents

Create `.agent.md` files that define specialized agents with specific roles, tool restrictions, and knowledge domains.

## Exercise 3: Skills

Build reusable `SKILL.md` files that package domain-specific knowledge for Copilot to invoke on demand.

## Exercise 4: Hooks

Set up pre/post event hooks that trigger Copilot actions automatically during your development workflow.

## Exercise 5: MCP Integration

Connect external tools and services via the Model Context Protocol, extending Copilot's capabilities beyond the local codebase.
